== Description ==

Website Chat Button: Kommo integration WordPress plugin provide access to [unified inbox](https://www.kommo.com/unified-inbox/) via button.

Use this plugin you agree to our [Terms of Service](https://www.kommo.com/terms/) and [Privacy Policy](https://www.kommo.com/privacy/)

Kommo's website chat button provides a convenient way for visitors to contact businesses using their preferred messaging app. Thanks to our seamless WordPress integration, visitors can directly reach out from the website to ask questions or share feedback. All conversations land in Kommo's unified inbox, making it effortless to manage and respond to them in no time.

To enhance the chat button experience, you can go a step further by adding a welcoming message, chatbots, and templates for automated replies. You also have the flexibility to customize the style to align with your brand's unique identity perfectly. The setup process is straightforward, ensuring a smooth connection between businesses and customers across popular messaging platforms.

== Frequently Asked Questions ==

This plugin uses [Kommo's CRM system API](https://www.kommo.com/) for button setting parameters, receiving leads, and responding to messages that come through the chat on your website.

= Do I need a Kommo account to use the website chat button? =
Yes. If you’re interested in trying out the website chat button, you can sign up for Kommo’s 14-day free trial, no credit card required.
During your free trial, you can access all of Kommo's features. Once the trial ends, the chat button will only be available if you subscribe to one of Kommo’s paid plan. Users on the Advanced or Enterprise plans get extra chat button capabilities. For example, you can set the button to show messages for visitors after a certain time on your site.

= What happens when my Kommo trial period ends? =
The chat button will remain active on your website, but you won’t be able to read or reply to your messages until you purchase a paid subscription plan. Kommo’s plans start at $15 USD per user/month.
Get more information on our plans here.

= What kind of messaging channels can I add to the website chat button? =
You can chat with clients seamlessly across all your favorite channels, including WhatsApp, Messenger, Instagram, Telegram, and many more. If you don’t use any of these channels to connect with clients, you can always use live chat.

= Can I customize the website chat button? =
Yes. On top of connecting your favorite messaging platforms, there are many ways to customize the chat button to match your brand’s identity. You can experiment with adding a welcome message, adjusting the button’s size, style, and placement, displaying relevant information about your team to clients, and much more.

= How can Kommo help me manage the messages I receive? =
Managing conversations is easy with Kommo's Unified inbox. All messages from every channel land in one place. This keeps all your customer communication centralized so nothing gets missed. From there, you can track your leads as they move through your sales process using Kommo's Digital pipeline.
Kommo also makes following up on leads simple. Use templates for quick responses. Or set up chatbots to handle routine tasks for your team. This lets you focus on what matters most - connecting with clients.

= Is there a limit to how much I can use the website chat button?=
No. You can receive, read, and reply to an unlimited number of messages through the chat button.

== Installation ==

= Automated Installation =

- Go to your WordPress admin dashboard, then find the Plugins menu.
- Click on Add New and search for website chat button.
- Activate the website chat button plugin through the Plugins menu in WordPress.
- Click the website chat button menu on the left.
- Create a new account or sign in if you already have a Kommo account.

= Manual Installation =

- Upload the plugin .zip file to the WordPress plugins directory (/wp-content/plugins/).
- Activate the website chat button plugin through the Plugins menu in WordPress.
- Click the website chat button menu on the left.
- Create a new account or sign in if you already have a Kommo account.
